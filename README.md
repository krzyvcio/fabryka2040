# fabryka2040

## Wymagania

- Bun
- LM Studio uruchomione na `http://localhost:1234/v1`
- Załadowany model (np. `lm-provers/QED-Nano-Q5_K_M`)

To install dependencies:

```bash
bun install
```

Lub (instalacja paczek AI ręcznie):

```bash
bun add ai @ai-sdk/openai
```

To run:

```bash
bun run start
```

Uruchomienie bezpośrednio debaty:

```bash
bun run debate
```

This project was created using `bun init` in bun v1.3.5. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
